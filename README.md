# moviesite

## Dhurandhar Movie Website

A modern, SEO-optimized movie website for Dhurandhar (2025) starring Ranveer Singh and Akshaye Khanna.

### Features

- **Modern Design**: Professional movie website with cinematic dark theme
- **SEO Optimized**: Fully optimized for search engines with relevant keywords
- **Download System**: Interactive download button with countdown popup
- **Responsive**: Works perfectly on all devices (desktop, tablet, mobile)
- **Movie Information**: Complete cast, box office, OTT release details
- **Social Media Ready**: Open Graph and Twitter Card optimization
